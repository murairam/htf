
## Step 2: Configure API Keys (1 min)

Create a `.env` file in the `essenceAI` directory:

```bash
# Required
OPENAI_API_KEY=sk-your-key-here

# Optional (but recommended for better data)
TAVILY_API_KEY=tvly-your-key-here

# LLM Provider
LLM_PROVIDER=openai
```

### Get API Keys:

**OpenAI** (Required - You have credits!):
- Use your existing OpenAI API key

**Tavily** (Optional - Free tier):
1. Go to https://tavily.com
2. Sign up (free)
3. Get API key (1000 searches/month free)

## Troubleshooting

### "OPENAI_API_KEY not found"
- Make sure you created the `.env` file in the `essenceAI` directory
- Check that your API key is correct

### "No PDF files found"
- PDFs should be in `essenceAI/data/` folder
- They're already there! If not, run: `cp ../hackthefork/*.pdf data/`

### "Module not found"
- Make sure you installed dependencies: `pip install -r requirements.txt`
- Consider using a virtual environment

### Tavily API not working
- It's optional! The app will fall back to OpenAI-only mode
- You'll still get competitor data, just generated by GPT-4o
